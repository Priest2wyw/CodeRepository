========================================================================
    控制台应用程序：chapter13 项目概述
========================================================================

应用程序向导已为您创建了此 chapter13 应用程序。

本文件概要介绍组成 chapter13 应用程序的每个文件的内容。


chapter13.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

chapter13.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

chapter13.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 chapter13.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

StdAfx.h中保存了二叉搜索树，AVL，红黑树的类声明，及节点的结构声明。
/////////////////////////////////////////////////////////////////////////////
其他注释:

应用程序向导使用“TODO:”注释来指示应添加或自定义的源代码部分。

/////////////////////////////////////////////////////////////////////////////
binary_search_tree.cpp描述了如何构建二叉搜索树，对与元素可以充分的二叉搜索树如何插入和删除
binary_balance_tree.cpp描述了AVL树的插入与删除操作
RedBlackTree.cpp描述了如何构建红黑树，及如何插入和删除的操作
